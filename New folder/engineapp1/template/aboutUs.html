WHAT IS PEAR?

Pear is a web application that aims to assist users in their brainstorming processes. With the help of a random word generator, users manually associate their given word with similar word. Upon the entry of each word, a thought map will appear, which will help users manually associate their given word with a similar word. Upon the entry of each word, a thought map will be rendered, which will help the user visually see their thought process. After pairing multiple words, the user will gradually arrive upon a brainstorming idea.
